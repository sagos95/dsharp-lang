# Change Log

All notable changes to the "dsharp-language" extension will be documented in this file.

## [0.1.0] - 2025-11-14

### Added
- Initial release
- Syntax highlighting for D# language (`.ds` files)
- Support for `func` keyword (replaces `void`)
- Support for `async func` keyword (replaces `async Task`)
- Full C# syntax support (strings, numbers, comments, keywords, types)
- String interpolation highlighting
- Bracket matching and auto-closing
- Code folding support
- Comment support (line and block)
- Preprocessor directive highlighting
